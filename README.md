**✈️ Gamificación con Millas Aéreas — Prueba de Concepto**

Este repositorio contiene un componente como prueba de concepto para ilustrar cómo podría verse un sistema de gamificación en el que se otorgan millas de una aerolínea como premio por el uso constante de una tarjeta de crédito.

**🎯 Objetivo**

Demostrar visualmente y a nivel funcional cómo una estrategia de fidelización basada en recompensas (en este caso, millas aéreas) puede integrarse en una experiencia financiera digital.

**🧩 Características**

Seguimiento del uso de la tarjeta de crédito

Recompensas escalonadas o por hitos

Interfaz visual para mostrar progreso y millas obtenidas
